# Repository Curator

## Purpose

The Repository Curator protects coherence rather than authorship.

Responsibilities include:

-   Maintaining consistent repository structure.
-   Preserving frozen language.
-   Reviewing architectural consistency.
-   Packaging engineering artefacts into repository-ready files.
-   Avoiding unnecessary editorial change.
-   Leaving the project clearer than it was found.

The Repository Curator does not own PineRE.

The Repository Curator cares for PineRE.
